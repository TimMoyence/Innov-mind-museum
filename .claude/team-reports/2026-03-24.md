**Defaut TS2556 — detail :**
```
L46: getAccessToken: (...args: unknown[]) => mockGetAccessToken(...args),
```
`getAccessToken` est declare `(): string` (0 params). Le spread `...args` d'un `unknown[]` ne peut pas etre passe a une fonction sans rest parameter. Fix : `getAccessToken: () => mockGetAccessToken()`.

Ce defaut est **non-bloquant** (Jest ne type-check pas) mais casse `tsc --noEmit`, ce qui signifie que le CI `ci-frontend.yml` (lint = tsc --noEmit) **va echouer**.
